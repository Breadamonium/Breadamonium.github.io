Afya Volunteering Sign-in | Web App

CS 4320 | BCNF Tree

October 2015

For this project, we implemented a B+ tree in java. Given a generic framework of a linkedlist of nodes, we had to properly implement insertion and deletion of records. Additionally, we had to follow the post condition in which we must split full buckets after an insertion, and then redistribute records to the left or right tree depending on the load. For deletion, we had to check if the bucket was underfull, and then combine buckets and redistribute if neccessary. Finally we had to reset the root of the tree if a split or a combine of buckets caused the tree level to increase or decrease. In this project, I learned about indexing and tree traversal for B+ trees.

Github is private for Academic integrity reasons.

CS 3410 | Network Honeypot

April 2015

We wrote a device driver in C for a simulated network card. We would be given packets of data in the form of bits. We had to process these bits to analyze the bits and classify them as spammer, normal, or evil. We were given 8 rings to act as 8 simulated threads running the same program concurrently, but with all 8 rings drawing from the same queue of packets. In this project, I learned about multi-threading and bit analysis.

CS 1110 | Breakout Game

December 2013

In this project, I created an imitation of the classic arcade game Breakout in python. Here we were given a library of classes including the ball, the bricks, the pad, along with given mouse-click and movement classes. I had to code up the physics of the ball movement after bouncing off the pad along with the actual mouse clicking and tracking to move the pad. Finally, the program had to check for when the player won or if he/she lost when time was up. In this project, I learned to read basic documentation of pre-written code/libraries and how to integrate other people's code with mine.
